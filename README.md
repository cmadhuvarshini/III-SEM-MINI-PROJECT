# III-SEMESTER-MINI-PROJECT
E-RESERVATION SYSTEM: FROM BOOKING TO BOARDING THE ROLE OF  TRAVEL AGENCIES

OBJECTIVE:

The e-reservation system developed through this project effectively 
addresses the complexities associated with flight booking and user 
management within the travel industry. By integrating essential 
features such as user registration, flight reservations, itinerary 
management, and package viewing, the system provides a 
comprehensive platform that facilitates efficient travel planning for 
users.
